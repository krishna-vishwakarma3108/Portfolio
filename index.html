Amazon Clone

Tic Tac Toe

Tic Tac Toe is a classic two-player game played on a 3x3 grid where players take turns placing X or O in empty cells. The goal is to be the first to get three of their symbols in a row, column, or diagonal. The game promotes strategic thinking and pattern recognition. It is easy to learn, making it a popular introductory game for children and a common exercise in programming for learning basic algorithms.
